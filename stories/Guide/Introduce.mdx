## ✨ 特性

- 📦 开箱即用的 React 组件
- 🔥 基于 antd 封装的业务组件
- 🛡 使用 TypeScript 开发，提供完整的类型定义文件

## ⌨️ 本地开发

```bash
$ git clone https://github.com/inbeen/binc.git
$ cd binc
$ pnpm install
$ pnpm storybook
```
